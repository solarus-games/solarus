The format of treasures has changed since r1410 to make them quest-dependent.
Now, the inventory items, the equipment items, the quest status items, the dungeon items,
the treasures, the chests, the shop items and the pickable items
are all described in a data file instead of being hardcoded.
This allows each quest to create its own items.

However, the data files of maps needs to be updated because the syntax of
chests, shop items, pickable items, destructible items and enemies has changed.
